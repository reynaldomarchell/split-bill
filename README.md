## Download

1. Clone this project

```bash
  git clone https://github.com/reynaldomarchell/split-bill.git split-bill
```

2.  Go to the project directory

```bash
  cd split-bill
```

3. Install the dependencies

```bash
  npm install
```

4. Start the app

```bash
  npm start
```
